    Rigid contact
      • Example

    rigid contact in equations of motion can be inforced by a single quadratic equality constraint and two linear inequality constraint \[\begin{aligned} \bm{ϕ}(\bm{q})⋅\bm{f} &= 0 \\ \bm{\phi}(\bm{q}) &≥ 0 \\ \bm{f} &≥ 0 \end{aligned}\] which can be written in the compact form of an LSP as \[\operatornamewithlimits{find}_{\bm{f}} \ \text{s.t.}: 0 ≤ \bm{ϕ}(\bm{q}) ⟂ \bm{f} ≥ 0\]

    Example #

    Take a simple cart Its equations of motion can be written as \[m\ddot{q} = u + f\] suplemented by contact constraints \[ fq = 0 \;,\quad q ≥ 0 \;,\quad f ≥ 0\] written in the compact form of an LSP as \[\operatornamewithlimits{find}_{f} \ \text{s.t.}: 0 ≤ q ⟂ f ≥ 0\]

    By discretizing the EoM we can approximate \( q_{n+1} \) using the semi-implicit Euler scheme \[\begin{aligned} q_{n+1} &≈ q_n + h \dot{q}_{n+1} \\ \dot{q}_{n+1} &≈ \dot{q}_n + h \ddot{q}_n \end{aligned}\] where \[\ddot{q}_{n} = \frac{u_n+f_n}{m}\] After consecutive substitutions we attain the approximation \[q_{n+1} = q_n + h \dot{q}_n + \frac{h^2}{m} u_n + \frac{h^2}{m} f_n\] for which we may formulate the LCP \[\operatornamewithlimits{find}_{f} \ \text{s.t.}: 0 ≤ q_{n+1} ⟂ f_n ≥ 0\]

    In line with the original definiton of the LCP we have \[\bm{w} \coloneqq q_{n+1} \;,\quad \bm{z} \coloneqq f \] and terms \[\begin{aligned} \bm{q} &\coloneqq q_n + h \dot{q}_n + \frac{h^2}{m} u_n \\ \bm{M} &\coloneqq \frac{h^2}{m} \end{aligned}\]

    More on the topic can be read in B3.3 of Russ Tedrake’s underactuated notes.
